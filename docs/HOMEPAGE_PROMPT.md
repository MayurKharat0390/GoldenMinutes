# Golden Minutes - Home Page Design Prompt

**Project Name:** Golden Minutes
**Type:** National Emergency Response Network (Decentralized)
**Target Audience:** Citizens, Volunteers, Government Officials.
**Vibe:** Professional, Trustworthy, High-Tech, Urgent, Modern.

## Structure & Elements Checklist

### 1. Navigation Bar (Command Center)
- **Logo:** "Golden Minutes" (Symbol: Shield/Pulse/Heartbeat).
- **Links (Anonymous):** Login, Register, "Join Force" (CTA).
- **Links (Logged In):** "Live Grid" (Map), "Dashboard", Profile, Logout.
- **Style Note:** Needs to look like a modern web app header. Clear hierarchy.

### 2. Hero Section (Above the Fold)
- **Headline:** Something powerful like "Minutes Matter" or "The Noblest Act is to Save a Life."
- **Sub-headline:** Explaining the value proposition: "We connect citizens with critical medical aid in milliseconds using a decentralized volunteer grid."
- **Primary CTA:**
    - For Visitors: "Join the Network" / "Become a Volunteer" (Prominent).
    - For Users: "Launch Command Grid" (Access Map).
- **Visuals:** Needs a high-impact visual (Map visualization, Heroic imagery, or Abstract tech waves).

### 3. Impact Dashboard (Live Stats)
- **Data Points to Display:**
    1.  **Missions Activated:** (e.g., 1,240+)
    2.  **Lives Impacted/Saved:** (e.g., 892)
    3.  **Avg. Response Time:** (e.g., 6.5 mins - *The "Golden" Metric*)
- **Style:** These numbers should be big, bold, and feel "live" or dynamic.

### 4. How It Works (The Protocol)
- **3-Step Process Cards:**
    1.  **Signal (SOS):** Victim triggers alert -> Geolocation broadcast.
    2.  **Deploy:** Nearby volunteers receive notification & navigation.
    3.  **Stabilize:** Volunteer arrives & provides CPR/First Aid before ambulance.
- **Icons:** Needs clear iconography for each step (Broadcast Tower, Location Pin, Heart/Medical).

### 5. "Why Trust Us" / Pillars
- **Key Features:**
    - **Hyper-Local:** Help comes from neighbors (2km radius).
    - **Verified Trust:** All volunteers are ID-verified.
    - **Secure:** Encrypted data & government compliant.
    - **Fast:** Beats ambulance traffic times.

### 6. Floating SOS Button (Critical)
- **Placement:** Fixed bottom-right or accessible area.
- **Look:** Must be the most dominant element. Red, Pulsating, "Press for Help".
- **Condition:** Only active for registered/verified users, but visually prominent in design concepts.

### 7. Call to Action (Footer Area)
- **Message:** "Ready to serve your community?"
- **Button:** "Enlist Today" (Register).
- **Social Proof:** "Join 5,000+ Active Responders".

### 8. Footer
- **Links:** Legal, Privacy, Contact, "For Hospitals", "For Government".
- **Helpline:** "24/7 Support: 1800-GOLD-SOS".
- **Badges:** "Approved for Civilian Use", "Secure".
